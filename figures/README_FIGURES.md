## How to Use

### For PNG Images:
1. Open the PNG file
2. Copy/paste directly into your document
3. Or use "Insert Image" in your document editor

### For Text Files (Screenshots):
1. Open the `.txt` file in a text editor (Notepad, VS Code, etc.)
2. Take a screenshot of the terminal/text output
3. Insert the screenshot into your document

### Recommended Screenshot Tools:
- **Windows**: Snipping Tool, Windows + Shift + S
- **VS Code**: Right-click file → "Open Preview" → Screenshot
- **Terminal**: Use terminal with monospace font for better appearance
